# Ecommerce Project

Este é um projeto de ecommerce utilizando **Laravel** no backend, **React** no frontend e **MySQL** como banco de dados. O projeto está dockerizado, facilitando o ambiente de desenvolvimento.

## Pré-requisitos

Antes de começar, verifique se você tem os seguintes pré-requisitos instalados:

- **Docker**: [Instalar Docker](https://www.docker.com/get-started)
- **Docker Compose**: [Instalar Docker Compose](https://docs.docker.com/compose/install/)
